Waartaa v0.2
============

**Waartaa** is currently a *IRC Client as a Service*. **Waartaa** is a word of
Indian origin meaning *to communicate*. Waartaa aims to become an intuitive open
source communication and collaboration tool. The code is open-source, and
`available on github <https://www.github.com/waartaa/waartaa>`_.
