Waartaa
=======

**Waartaa** is *IRC Client as a Service*. **Waartaa** is a word of Indian origin
meaning *to communicate*. Waartaa is a communication tool for developers,
unlike it's proprietary counterparts, Waartaa respects user privacy and want
to give user full control of their data. Waartaa is currently in ``0.1-rc-1``
release. The code is open-source, and `available on github
<https://www.github.com/waartaa/waartaa>`_.
